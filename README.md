# roblox-silicon
roblox-silicon is a quick bootstrapper for macOS that gives Roblox increased performance by using the native macOS rendering library (Metal).
This isn't a whole huge project, it's just a quick patcher I pulled together. I'm open to PRs if anyone else has some cool tweaks to increase M-series performance.

## Instructions
1. Download the roblox-silicon Installer from the [releases page](https://github.com/sirkingbinx/roblox-silicon/releases/latest)
2. Open the app
3. On the dropdown, select the latest version available
4. Select "Apply Patch"
5. You're done. Launch roblox-silicon from the launchpad to use the bootstrapped version.
